# daily-coding-python-simple
Simple daily codes for practice
First file: My Frist game. I aimed to make a Magic 8 Ball like game to work on basics. How it works is that you ask a YES or NO question and it responds with a random answer like Yes, No, or maybe. The code runs randomly by using the random function and picking random numbers out of 3. And, for each integer in the group it has a response. Additionally, if the user says to not continue it stops the program.
